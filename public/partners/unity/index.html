// how it all works together

Why these Unity solutions are the ultimate combo.

Unity’s suite of tools—Industry License, Asset Manager, Asset Transformer SDK and Studio—work together to streamline every stage of a real-time 3D project, from asset ingestion to deployment and collaboration. Companies can eﬃciently manage, convert, optimize, and deploy complex 3D data, no matter the industry.

STEP_01

Asset Transformer Studio

CAD Prep & Decimation: Import massive engineering datasets (JT, STEP, CATIA, FBX) through a visual UI. Decimate polygons, merge meshes, generate LODs, and clean hierarchies to prepare raw CAD assets for real-time visualization.

STEP_02

Asset Transformer SDK

Pipeline Automation: Embed the Transformer SDK into automated enterprise ingestion pipelines. Programmatically process engineering updates, apply batch optimizations, and output runtime-ready formats without manual editor overhead.

STEP_03

Asset Manager

Centralized Cloud Hub: Store, tag, version-control, and manage all optimized 3D assets in a single secure cloud repository. Allow developers, engineers, and designers across the globe to instantly discover, fetch, and collaborate.

STEP_04

Unity Industry

Enterprise Deployment: Compile and deploy interactive 3D applications, immersive VR training simulators, AR experiences, and web configurators. Deliver real-time visualizations across mobile, desktop, and headset devices.
